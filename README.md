# claude-monitor-web

Landing page for [claude-monitor](https://github.com/cjthompson/claude-monitor) — a terminal dashboard that monitors and auto-accepts Claude Code permission prompts across iTerm2 panes.

## Structure

- `index.html` — Single-page static site with inline CSS/JS
- `images/` — Screenshots used on the landing page

## Development

Open `index.html` in a browser. No build step required.

## Deployment

Static site — deploy to any static hosting (GitHub Pages, Netlify, Vercel, etc.).
