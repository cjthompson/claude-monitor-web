# claude-monitor-web

Landing page for the claude-monitor project.

## Project Structure

- Single static HTML page (`index.html`) with inline CSS and JS
- Screenshots in `images/`
- No build tools, frameworks, or dependencies

## Conventions

- All styles are inline in `<style>` tags within `index.html`
- CSS uses custom properties defined in `:root`
- Dark theme with monospace font (JetBrains Mono)
- Terminal/code aesthetic matching the claude-monitor tool itself
- GitHub repo: https://github.com/cjthompson/claude-monitor
